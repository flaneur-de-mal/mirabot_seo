Mirabot SEO
A Drupal 7 feature as a starting point for basic SEO.

-- About

Mirabot SEO uses Features to package up minimal SEO starting point for a new
Drupal 7 site.

-- Prerequisites

1. features (https://drupal.org/project/features)
2. See mirabot_seo.info for additional features dependencies.

-- Installation Instructions

1. Install and enable the features module.
2. Enable the mirabot_seo module, preferably using drush or another method
that will automatically enable subdependencies.
